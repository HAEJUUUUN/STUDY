STUDY

컴퓨터란? > 전자회로를 이용해 다양한 종류의 데이터를 처리하는 도구이다.
‘계산하다’라는 뜻의 라틴어에서 유래하였으며 최근 컴퓨터를 사용해 정보를 수집하고 분석하는데 데이터베이스 기술이 활용되고 있다.
컴퓨터 프로그램은 컴퓨터가 처리해야 하는 작업을 순서대로 모아놓은 것이다. 여기에서 처리하는 것은 데이터이다.
그리고 컴퓨터는 [입력 데이터 > 데이터 처리 > 출력 데이터] 이러한 환경에서 동작하는 기계이다.

데이터 =/= 정보. 데이터는 매체에 저장된 사실을 말하지만 정보는 데이터를 처리해서 사람이 이해라기에 적합한 형태로 의미 있게 만든 것이다.
데이터베이스 시스템의 목적은 정보와 데이터 사이의 갭을 줄이는 것이며 데이터는 프로그램과 질의에 의해 정보로 변환된다.

데이터베이스는 어떤 조직체에 관련된 사실들을 수집한 데이터이다. 이는 일반적으로 다수 사용자 혹은 다수 응용 프로그램들이 공유하기 위해 설계된다.
즉, 데이터베이스는 조직체의 응용 시스템들이 공유해서 사용하는 운영 데이터들이 구조적으로 통합되어 저장 장치에 저장된 형태의 데이터 모임이다.
데이터베이스는 현재 대학에서 학생들, 학과, 교수에 관한 정보를 유지하거나 스캐너가 바코드를 읽고 상품의 가격을 찾아내는 것처럼 실생활에 항상 존재하고 있다.
