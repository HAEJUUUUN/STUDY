jQuery 개요

존 레식에 의해 개발된 경량 javascript 라이브러리로 복잡한 javascript 코드를 손쉬게 구현하기위해 개발되었다.
1. DOM과 관련된 처리를 쉽게 구현할 수 있다.
2. ajax 통신, 이벤트 처리 등을 폭넓게 지원한다.
3. 별도의 플러그인을 통해 차트, 슬라이드 쇼, 테이블 등을 간단히ㅣ 구현할 수 있다. (부트스트랩에서 활용)

jQuery 연결

1. jQuery 홈페이지에서 js 파일을 다운로드 받아 연결하는 방법
2. CDN(Content Delivery Network)를 통해 연결하는 방법

$(document).ready

크롬에서 3.6 버전의 jQuery 라이브러리는 다운로드 받는 방식과 cdn방식 모두 window.onload 다음에 $(document).ready()가 실행된다. window.onload는 맨 마지막에 실행된다.
